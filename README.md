# Blog

Sistema web com funcionalidade de um blog comum, com: feed, paginção, pesquisa, filtragem de artigos e categorias. Também possui CRUD com banco de dados, tabelas com dados para categorias, artigos e administradores( usuários cadastrados, apenas admin ). O projeto ainda esta em crescimento e sendo melhorado.
